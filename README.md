# 📊 Sales Performance & Customer Insights Dashboard

## 🚀 Project Overview
This project presents a comprehensive **Power BI dashboard** designed to analyze **sales performance, customer behavior, and churn risk**. 

The dashboard transforms raw transactional data into **actionable business insights**, enabling stakeholders to:
- Monitor revenue trends
- Identify high-value customers
- Detect churn risks
- Evaluate retention strategies

---

## 🎯 Business Problem
Businesses often struggle to:
- Identify which customers generate the most value
- Understand purchasing behavior patterns
- Detect early signs of customer churn
- Evaluate effectiveness of retention strategies

This dashboard addresses these challenges by providing a **data-driven decision support system**.

---

## 📂 Dataset Description

The dataset contains customer transaction and behavioral data:

| Column | Description |
|------|-------------|
| Customer_ID | Unique identifier for each customer |
| Product_ID | Product purchased |
| Transaction_ID | Unique transaction record |
| Purchase_Frequency | Number of purchases made |
| Average_Order_Value | Average spend per order |
| Time_Between_Purchases | Gap between purchases |
| Region | Customer location |
| Churn_Probability | Likelihood of churn |
| Lifetime_Value | Total value generated by customer |
| Season | Seasonal purchase trend |
| Preferred_Purchase_Times | Time of purchase behavior |
| Retention_Strategy | Strategy applied to retain customer |

## 🛠 Tools & Technologies

- **Power BI** → Data visualization & dashboarding  
- **DAX (Data Analysis Expressions)** → KPIs and calculations

---

## ⚙️ Data Preparation & Modeling

- Cleaned dataset (handled nulls, ensured data consistency)
- Converted date columns into proper format
- Created **Date Table** for time-based analysis
- Established **relationships (Star Schema approach)**
- Created calculated columns:
  - Customer Segment
  - Churn Category
- Developed key measures:
  - Total Revenue
  - Avg Order Value
  - Avg Churn Rate
  - High Risk Customers

---

## 📊 Dashboard Structure

---

### 🔹 1. Executive Summary

Provides a high-level overview of business performance:

- Total Revenue  
- Total Customers  
- Average Order Value  
- Monthly Sales Trend  
- Revenue by Region  

📌 **Goal:** Enable quick business performance tracking

---

### 🔹 2. Customer Insights

Focuses on customer segmentation and behavior:

- Customer Segment Distribution  
- Customer Purchase Behavior (Scatter Plot)  
- Top Customers by Lifetime Value  
- Revenue Contribution by Segment  

📌 **Goal:** Identify high-value and loyal customers

---

### 🔹 3. Churn Analysis

Focuses on risk detection and retention:

- Average Churn Rate  
- High-Risk Customer Count  
- Churn Distribution  
- Churn Rate by Region  
- Retention Strategy Effectiveness  
- High-Risk Customer Table  

📌 **Goal:** Reduce churn and improve retention strategies

---

## 📈 Key Insights

- High-value customers contribute a disproportionate share of revenue  
- Customers with lower purchase frequency show higher churn probability  
- Certain regions exhibit significantly higher churn rates  
- Some retention strategies are less effective and require optimization
